Try the *409 helpline

Dial *409 using the keypad, press the green call button, then navigate the menu. This is exactly what a caller hears — in Tamil or English.

  • Press 1 — request food for your community
  • Press 2 — check your request status
  • Press 3 — switch between Tamil and English

How it works
Every component is free. Deployed and running in under 2 hours.
User dials *409
→
Twilio receives call
→
Flask on Render
→
Tamil / English IVR
→
Ticket saved in Supabase
→
Volunteer SMS sent
Twilio IVR
Handles the actual phone call. Plays Tamil or English voice using Polly TTS. Captures key presses (1/2/3) and sends them to your Flask server via webhook.
$15 free trial
Flask backend
Receives Twilio webhooks, runs the IVR logic, saves tickets to Supabase, triggers the matching engine, and sends SMS confirmations.
Free on Render.com
Supabase database
Stores all tickets, donors, and volunteer assignments. Provides real-time updates to the dashboard. Two tables: tickets and donors.
Free forever
Matching engine
Automatically pairs each food request with the nearest available donor based on capacity and urgency. Notifies the volunteer via SMS instantly.
Built into Flask app
Dashboard
Live command centre showing tickets, analytics, donor management, and volunteer status. Auto-refreshes every 15 seconds.
Served by Flask
Tamil+English IVR
Full bilingual support. Caller presses 3 to switch language mid-call. All prompts, confirmations and SMS messages are in Tamil or English.
